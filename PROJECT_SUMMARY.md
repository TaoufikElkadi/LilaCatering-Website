## 🆘 Need Help?

### Common Tasks

**Q: How do I add a new menu item?**  
A: Edit `/data/menuData.ts` and add a new object to the `menuItems` array.

**Q: How do I change colors?**  
A: Edit `tailwind.config.ts` under the `colors.primary` section.

**Q: How do I add my logo?**  
A: Place logo in `/public/images/` and update `/components/Navigation.tsx`.

**Q: How do I stop the dev server?**  
A: Press `Ctrl + C` in the terminal running the server.

**Q: Website not loading?**  
A: Make sure the dev server is running: `npm run dev`
