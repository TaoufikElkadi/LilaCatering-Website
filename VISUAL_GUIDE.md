# 🎨 Visual Guide - Lila Catering Website

## Color Palette

```
🖤 Primary Black    #000000  - Main backgrounds
🤍 Primary White    #FFFFFF  - Text on dark backgrounds
✨ Primary Gold     #D4AF37  - Accent color, buttons, highlights
🔴 Dark Red         #8B0000  - Secondary accent, gradients
🟤 Golden Brown     #996515  - Tertiary accent, borders
```

## Typography

**Headings (Serif)**: Playfair Display
- Display: 72px - 96px (Hero titles)
- H1: 56px - 72px
- H2: 40px - 56px
- H3: 32px - 40px

**Body Text (Sans-serif)**: Inter
- Large: 20px - 24px
- Regular: 16px - 18px
- Small: 14px - 16px

## 🎬 Animations

### Page Load
1. Navigation slides down
2. Hero content fades in + slides up
3. Section elements fade in on scroll

### Interactions
- **Hover**: Lift effect (-10px translate)
- **Click**: Scale down (0.95)
- **Navigation**: Smooth scroll to sections
- **Cart**: Slide up from bottom
- **Modal**: Fade in + scale up

### Scroll Effects
- Scroll indicator bounces
- Navigation becomes opaque
- Elements fade in as they enter viewport
- Parallax on hero background

## 🎨 Design Patterns

### Moroccan Pattern (Background)
```
Subtle geometric patterns
Opacity: 5-10%
Color: Gold (#D4AF37)
Used on: Services, Footer, Some sections
```

### Gold Gradient
```
Linear gradient: 135deg
From: #D4AF37 (Gold)
To:   #996515 (Golden Brown)
Used on: Buttons, Badges, Accents
```

### Text Gradient
```
Background clip text effect
Colors: Gold shades
Used on: Main headings
```

```
Mobile:     < 768px   (1 column layouts)

```
